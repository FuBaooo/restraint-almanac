### 干嘛的
和隔离食用手册类似，但是推荐食材是根据黄历来的，你冰箱里不一定有，你也不一定吃的到，纯属娱乐。

### 有啥逻辑吗？
根据当天黄历实时变动，然后根据关键字从数据中匹配。<br>
每个食材都是每个黄历关键词能发散联想出来的。<br>
例如: 

- 祭祀 -> 祭祀代表的食材
- 嫁娶 -> 婚礼食材 + 婚礼行为 (拼酒)
- 出行 -> 动物的迁徙习性

等等
